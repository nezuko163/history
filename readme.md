# History

Приложение, в котором пользователи могут состязаться в своих знаниях истории

## Стек
- Kotlin
- Jetpack compose
- Firebase Database
- Firebase Auth
- Yandex s3
- Amazon s3 sdk
- jetpack navigation
- Hilt
